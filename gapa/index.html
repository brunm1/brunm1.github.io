Architektur

Systemabgrenzung / Umgebung / externe Schnittstellen

Gapa verbindet sich auf einen Websocket-Server. Die Nachrichten werden über JSON an Gapa verschickt. Die Struktur der Nachrichten ist in einem JSON Schema definiert. Das Modul gapa-integration enthält Code, um Nachrichten von Java Objekten zu JSON und zurück zu konvertieren. Daher kann es auf Client- wie auch Server-Seite eingesetzt werden, um Nachrichten richtig codiert zu senden und beim Empfänger (Gapa) korrekt zu decodieren. Aktuellstes Readme des gapa-integration Moduls. Die Systemabgrenzung ist im Anforderungsdokument unter Systemabgrenzung genauer beschrieben.
